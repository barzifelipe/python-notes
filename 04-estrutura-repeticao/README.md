# Estruturas de Repetição

Exemplos de estruturas de repetição em Python.

## Estruturas

| Estrutura | Descrição                                                      |
| --------- | -------------------------------------------------------------- |
| for       | Repete um bloco de código um número determinado de vezes       |
| while     | Repete um bloco de código enquanto uma condição for verdadeira |

## Arquivos

* `for.py` → Demonstra o uso da estrutura `for`.
* `while.py` → Demonstra o uso da estrutura `while`.
* `tabuada.py` → Gera a tabuada de um número utilizando `for`.
* `contagem_regressiva.py` → Realiza uma contagem regressiva utilizando `while`.
* `soma_numeros.py` → Soma os números de 1 a 10 utilizando `for`.
