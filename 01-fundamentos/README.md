# Fundamentos

Exemplos dos conceitos básicos da linguagem Python.

## Conceitos

| Conceito           | Descrição                                        |
| ------------------ | ------------------------------------------------ |
| Variáveis          | Armazenam informações na memória                 |
| Input              | Recebe dados do usuário                          |
| Conversão de Tipos | Converte valores entre diferentes tipos de dados |

## Arquivos

* `variaveis.py` → Demonstra a criação e utilização de variáveis.
* `input.py` → Demonstra como receber dados do usuário.
* `conversao_tipos.py` → Demonstra a conversão entre strings, inteiros e números decimais.
