# Matplotlib.pyplot

Exemplos da biblioteca matplotlib.pyplot para visualização de dados.

## Funções

| Função | Descrição |
|---------|-----------|
| plot() | Cria gráficos de linha |
| bar() | Cria gráficos de barras |
| pie() | Cria gráficos de pizza |
| title() | Define o título do gráfico |
| xlabel() | Define o rótulo do eixo X |
| ylabel() | Define o rótulo do eixo Y |
| show() | Exibe o gráfico |

- grafico_barras.py → Exemplo de gráfico de barras.
