# Tratamento de Exceções

Exemplos de tratamento de erros e exceções em Python.

## Conceitos

| Conceito | Descrição |
|-----------|-----------|
| Tipos de Exceções | Erros gerados durante a execução do programa |
| Try / Except | Captura e trata exceções |
| KeyError | Erro ao acessar uma chave inexistente em um dicionário |
| Finally | Bloco executado independentemente da ocorrência de erros |
| Raise | Lança uma exceção manualmente |
| Exceções Personalizadas | Validação de regras utilizando raise |

## Arquivos

- `tipos_excecoes.py` → Exemplos de exceções comuns.
- `tratando_excecoes.py` → Demonstra o uso de try e except.
- `excecoes_dicionarios.py` → Tratamento de exceções em dicionários.
- `finally.py` → Demonstra o uso do bloco finally.
- `raise.py` → Demonstra como lançar exceções manualmente.
- `raise_personalizado.py` → Demonstra validações utilizando raise.
